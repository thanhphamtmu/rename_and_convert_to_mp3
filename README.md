# Media Renamer & MP3 Converter

Script Python để:

1. Tự động đổi tên file `.m4a`/`.mkv` theo định dạng `YYYY-MM-DD HH-MM-SS`.
2. Nếu prefix hiện tại khác với prefix mới thì sẽ cập nhật và đổi tên file `.mp3` tương ứng (nếu có).
3. Chuyển đổi các file âm thanh/video sang định dạng `.mp3` bằng `ffmpeg`.

## Yêu cầu

- Python 3.7+
- ffmpeg (cài đặt sẵn trong hệ thống)

## Cài đặt

```bash
pip install -r requirements.txt
```
